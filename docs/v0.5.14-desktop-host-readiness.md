# v0.5.14 Desktop Host Readiness

**状态**：Released
**日期**：2026-05-18  
**分支**：`codex/v0.5.14-desktop-host-readiness`

## 一句话判断

v0.5.14 是进入 Tauri tray-first Desktop 前的 Core 小版本：不做 Desktop、不做 daemon、不扩展 Router，而是给 Desktop CLI runner 一个稳定的 host handshake。

## 背景

v0.5.8 到 v0.5.13 已经补齐 Desktop alpha 的关键消费链路：

- Router MCP runtime。
- Safe Mount Apply / Rollback。
- runs list / inspect。
- `.skr import`。
- import -> switchboard -> exposure -> Router 的端到端合同测试。

现在 Desktop 可以开工，但托盘应用还缺一个极轻的 Core 自检入口。仅靠 `skillrun --version` 不够，因为 UI 还需要知道：

- 当前二进制版本。
- Manifest IR / IPC / Adapter Protocol 版本。
- 本机平台。
- SkillRun home 与 registry path。
- 哪些 Desktop-facing JSON surfaces 可消费。
- Router 是否应由 Desktop 托盘作为 daemon 启动。

## 范围

### 新增

```text
skillrun host status --json
```

返回 `host.status.v1`：

- `binary.name/version`。
- `contracts.manifest_ir_version`。
- `contracts.ipc_protocol_version`。
- `contracts.adapter_protocol`。
- `platform.os/arch/family`。
- `paths.current_exe/skillrun_home/registry_path`。
- Desktop-facing `capabilities` 列表。
- Desktop 边界声明：
  - 必须消费 CLI JSON。
  - 不读取 Core 内部路径。
  - 不把 Router 当作托盘 daemon 启动。

### 测试

- `tests/fixtures/contracts/host_status.json` 冻结 host status JSON contract。
- `tests/consumer_json_contracts.rs` 覆盖 host status fixture。
- `tests/cli.rs` 确认 help surface 包含 `host`。

## 非目标

v0.5.14 不做：

- Desktop / Tauri / `skillrun ui`。
- daemon API。
- Router hot reload。
- Router process manager。
- HTTP / SSE / local socket API。
- multi-client mount apply。
- signed trust。
- dependency installer。
- package registry / marketplace。
- artifact/log/input content viewer。

## Desktop 使用方式

Desktop 启动时可以先调用：

```text
skillrun host status --json
```

然后根据返回结果判断：

- Core 是否可启动。
- 二进制版本是否满足 Desktop 最低要求。
- 当前平台与路径。
- 哪些短跑命令可用于 tray refresh。
- Router 应由 MCP client 配置启动，而不是 Desktop 托盘后台默认启动。

## 验收标准

- `skillrun host status --json` 成功返回 `host.status.v1`。
- 输出不读取 capsule、不读取 `.skillrun/runs`、不解析 Manifest YAML、不启动 Router。
- Host status fixture 在本地与 CI 中可跨平台归一化。
- README / docs index / release notes 说明 v0.5.14 是 Desktop Host Readiness，不是 Desktop release。
